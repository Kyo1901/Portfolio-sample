## 📌 1. 서비스 기본 정보

### 서비스 이름
**맛ZIP (MatZIP)** 🏠✨

### 서비스 주제
**지역 기반 맛집 리뷰 & 추천 플랫폼**
- 내 주변 맛집을 찾고, 신뢰할 수 있는 리뷰를 공유하는 커뮤니티

### 핵심 가치
```
✨ 진짜 경험담만 담은 리뷰
📍 위치 기반 맞춤 추천
👥 신뢰할 수 있는 커뮤니티
📸 한눈에 보는 비주얼 리뷰
```

**차별화 포인트:**
- 방문 인증 시스템으로 신뢰도 향상
- AI 기반 취향 분석 및 맞춤 추천
- 친구와 맛집 리스트 공유 기능

### 타겟 사용자
- **주요 타겟:** 20-40대 직장인 및 대학생
- **특징:** 외식을 즐기고 SNS 활동이 활발한 사용자
- **니즈:** 믿을 수 있는 맛집 정보, 빠른 검색, 사진 중심의 리뷰

## 📱 5. 모바일 UI 특징

### 네비게이션 구조

```
┌─────────────────────────┐
│     상단 고정 헤더       │  ← 로고, 알림, 검색
├─────────────────────────┤
│                         │
│      메인 콘텐츠         │
│                         │
│                         │
│                         │
├─────────────────────────┤
│  [홈] [검색] [+] [MY]   │  ← 하단 탭 네비게이션
└─────────────────────────┘
```

**네비게이션 타입:**
- **하단 탭 바** (Bottom Tab Navigation) - 메인
  - 홈, 검색, 작성(+), 마이페이지
  - iOS/Android 표준 높이 준수
  - 아이콘 + 라벨 조합

- **상단 헤더** (Top Header) - 보조
  - 로고 (홈에서만)
  - 페이지 타이틀
  - 뒤로가기 버튼
  - 액션 버튼 (공유, 더보기 등)

### 주요 버튼 위치

#### Primary Actions (주요 행동)
- **플로팅 버튼** (FAB)
  - 위치: 우측 하단
  - 용도: 리뷰 작성 (상세 페이지)
  - 크기: 56x56px
  - 색상: 메인 오렌지

- **CTA 버튼**
  - 위치: 하단 고정 (상세 페이지)
  - 용도: 전화하기, 길찾기
  - 높이: 48px
  - 풀 너비

#### Secondary Actions (보조 행동)
- 북마크: 우측 상단
- 공유하기: 우측 상단
- 필터: 상단 바

### 터치 영역 크기

**Apple Human Interface Guidelines 준수**
- 최소 터치 영역: **44x44pt** (iOS)
- 권장 터치 영역: **48x48dp** (Android Material Design)
- 버튼 간격: 최소 8px

**적용 사례:**
```
✅ 좋은 예시
┌──────────┐  ┌──────────┐
│  버튼1   │  │  버튼2   │  ← 48px 높이, 8px 간격
└──────────┘  └──────────┘

❌ 나쁜 예시
┌────┐┌────┐
│버튼││버튼│  ← 너무 작고 붙어있음
└────┘└────┘
```

### 세로/가로 모드

**기본 방향:** 세로 모드 (Portrait)
- 모든 페이지 세로 모드 최적화
- 가로 모드는 선택적 지원

**가로 모드 지원 페이지:**
- 사진 갤러리 보기
- 맛집 지도 페이지

**고정 방향:**
- 리뷰 작성 페이지 (세로 모드 고정)
- 설정 페이지 (세로 모드 고정)
